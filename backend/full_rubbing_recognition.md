# 甲骨文全文拓片识别 + AI推断系统技术方案

## 一、系统概述

**目标**: 上传整张甲骨拓片 → 输出全文内容 → 未知字用AI推断可能含义

## 二、技术可行性

### ✅ 现有成果支持

1. **厦门大学 + 腾讯 "殷契行智" (2025)**
   - 全球首个甲骨文智能体
   - 功能: 拓片识别 + 释义查询 + 文献溯源
   - 已面向公众开放

2. **复旦大学 可解释破译框架 (2025)**
   - 基于Qwen2.5-VL-7B大模型
   - 部首-象形双重匹配机制
   - 可输出可解释性分析文本
   - SOTA准确率

3. **华中科技大学 OBSD (ACL 2024)**
   - 条件扩散模型破译未知甲骨文字
   - 零样本学习，可处理未见过的字符
   - 生成现代汉字作为破译线索

4. **YOLOv8 拓片分割识别 (2024 MathorCup)**
   - mAP50: 0.928 (目标检测)
   - Top1_acc: 0.897 (字符分类)
   - 195种字符识别

## 三、系统架构设计

```
输入: 甲骨拓片图像
        ↓
┌─────────────────────────────────────┐
│ 阶段1: 单字分割 (YOLOv8)             │
│ - 检测拓片中的每个单字位置           │
│ - 输出: 每个字的bounding box        │
│ - 推理时间: ~120ms/字               │
└─────────────────────────────────────┘
        ↓
┌─────────────────────────────────────┐
│ 阶段2: 单字识别 (ResNet50/MobileViT) │
│ - 对每个单字进行分类识别            │
│ - 已知字 → 输出对应现代汉字         │
│ - 未知字 → 标记为 [未释读]          │
│ - 准确率: 80-90%                    │
└─────────────────────────────────────┘
        ↓
┌─────────────────────────────────────┐
│ 阶段3: AI推断未知字 (LLM)           │
│ - 输入: 未知字的图像 + 上下文        │
│ - 使用: Qwen2.5-VL / GPT-4V         │
│ - 分析: 部首 + 象形 + 上下文语义     │
│ - 输出: 推断含义 + 可信度           │
└─────────────────────────────────────┘
        ↓
┌─────────────────────────────────────┐
│ 阶段4: 全文整理 + 释义               │
│ - 按阅读顺序排列识别结果            │
│ - 提供逐字释义和句意翻译            │
│ - 生成完整释文                      │
└─────────────────────────────────────┘
        ↓
输出: 拓片全文 + 逐字释义 + AI推断结果
```

## 四、核心技术栈

### 4.1 单字分割 (检测)
```python
# YOLOv8 检测模型
from ultralytics import YOLO

model = YOLO('yolov8s.pt')
model.train(data='oracle_detection.yaml', epochs=100)

# 检测单字位置
results = model.predict('rubbing.jpg')
bboxes = results[0].boxes.xyxy.tolist()
```

### 4.2 单字识别 (分类)
```python
# 方案A: ResNet50 (简单快速)
from torchvision.models import resnet50
model = resnet50(pretrained=True)
model.fc = nn.Linear(2048, num_classes)

# 方案B: MobileViT-S (轻量级, 80%+准确率)
import timm
model = timm.create_model('mobilevit_s', pretrained=True)
```

### 4.3 AI推断未知字 (大模型)
```python
# 方案1: 使用Qwen2.5-VL (开源, 可本地部署)
from transformers import Qwen2VLForConditionalGeneration, AutoProcessor

model = Qwen2VLForConditionalGeneration.from_pretrained(
    "Qwen/Qwen2.5-VL-7B-Instruct"
)

# 推断提示词
prompt = """
请分析这个甲骨文未知字符：
1. 观察字形，分析可能的部首构成
2. 根据象形特征推断可能的含义
3. 结合上下文语境给出最可能的现代汉字对应
4. 给出推断的可信度（0-100%）

请输出结构化分析结果。
"""
```

## 五、创新亮点 (适合比赛)

### 1. 全文识读 (vs 单字识别)
- 用户上传整张拓片
- 系统自动分割、识别所有单字
- 按顺序输出完整释文

### 2. AI推断未知字 (vs 仅识别已知字)
- 未知字不再显示"无法识别"
- 利用大模型分析部首、象形、上下文
- 给出推断含义和可信度
- 提供可解释性分析过程

### 3. 多模态交互
- 拍照上传拓片 → 实时识别
- 点击任意单字 → 查看详细信息
- AI推断结果可人工校正 → 数据反哺模型
